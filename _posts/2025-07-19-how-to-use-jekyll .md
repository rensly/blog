[官方教程](https://docs.github.com/zh/pages/quickstart)

主要步骤：
1. 建立git仓库
2. 配置pages环境
3. 本地建立jekyll项目
4. 编辑博客预览
5. 上传

常用命令：
```
# 安装Jekyll和Bundler
gem install jekyll bundler

# 创建新的Jekyll站点
jekyll new my-blog

# 进入站点目录
cd my-blog

# 启动本地服务器
bundle exec jekyll serve
```
